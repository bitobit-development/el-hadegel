# Knesset Members of the 25th Knesset

## Overview

This directory contains information about all current Members of Knesset (MKs) for the 25th Knesset of Israel.

**Source:** [Knesset Official Website](https://m.knesset.gov.il/mk/apps/mklobby/main/current-knesset-mks/all-current-mks)

**Data captured:** November 25, 2025

**Total Members:** 120 MKs

## Files in this directory

- `README.md` - This overview file
- `all-mks-list.md` - Complete list of all 120 MKs with their parties and profile links
- `parties-breakdown.md` - Breakdown of MKs by political party/faction

## Key Information

The 25th Knesset (Hebrew: הכנסת ה-25) consists of 120 members representing various political parties and factions.

### Main Political Factions Represented:

- **Likud (הליכוד)** - The ruling party
- **Yesh Atid (יש עתיד)** - Opposition party
- **National Unity (כחול לבן - המחנה הממלכתי)** - Blue and White
- **Shas (התאחדות הספרדים שומרי תורה)** - Ultra-Orthodox Sephardic party
- **United Torah Judaism (יהדות התורה)** - Ultra-Orthodox Ashkenazi party
- **Religious Zionist Party (הציונות הדתית)** - Religious nationalist party
- **Jewish Power (עוצמה יהודית)** - Right-wing party led by Itamar Ben Gvir
- **Yisrael Beiteinu (ישראל ביתנו)** - Secular right-wing party
- **Ra'am (רע"ם)** - Arab party
- **Hadash-Ta'al (חד"ש-תע"ל)** - Arab joint list
- **Labor (העבודה)** - Social democratic party
- **Noam (נעם)** - Religious conservative party

## Data Fields

Each MK entry includes:
- **ID**: Unique Knesset identifier
- **Name**: Full name in Hebrew
- **Party/Faction**: Political affiliation
- **Profile URL**: Link to official Knesset profile page
- **Photo**: Official portrait photo URL
